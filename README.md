# Personal Projects

This repository contains personal electrical and software engineering projects focused on embedded systems, PCB design, and hardware integration.

Most of these projects are built from the ground up, including:
- Firmware development (ESP32, Arduino framework)
- Custom PCB schematic and layout in KiCad
- Hardware debugging and system integration
- 3D printed mechanical design
- Power management and battery systems

The goal of these projects is practical system implementation — taking a concept from idea to working hardware.

---

## Projects

### ESP32 MP3 Jukebox
Custom embedded audio system built around an ESP32 with a fully designed PCB, firmware, and enclosure.

Includes:
- ESP32 firmware
- Custom KiCad schematic and PCB layout
- Gerber manufacturing files
- Mechanical STL files
- System documentation

Project folder: `MP3 Jute Box/`
